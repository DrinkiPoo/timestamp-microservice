Timestamp Microservice

What is the purpose of this API?

It gives you unix timestamps!

How do I use it?

like this: projectURL/api/2001-06-01

or: projectURL/api/991353600000


What's the output like?

{"unix":991353600000, "utc":"Fri, 01 Jun 2001 00:00:00 GMT"}
